## Overview

This project explores Python’s core data structures and their practical usage in a game-like data processing environment.  
It focuses on learning how to efficiently store, process, and analyze data using Python collections such as lists, tuples, sets, dictionaries, generators, and comprehensions.

---

## What I Learned

### Exercise 0 — Command Quest

- Using `sys.argv` to read command-line arguments
- Understanding how programs receive external input
- Working with lists and indexing
- Handling argument counting and formatting output

---

### Exercise 1 — Score Cruncher

- Handling user input and errors with `try/except`
- Storing and processing numerical data in lists
- Using built-in functions:
  - `sum()`
  - `max()`
  - `min()`
  - `len()`
- Calculating:
  - total
  - average
  - highest and lowest values
  - range
- Filtering valid and invalid inputs

---

### Exercise 2 — Position Tracker

- Using tuples to store fixed data (3D coordinates)
- Validating user input with retry logic
- Working with `math.sqrt()` for distance calculations
- Understanding immutability of tuples
- Applying the Euclidean distance formula

---

### Exercise 3 — Achievement Hunter

- Using sets to store unique values
- Performing set operations:
  - `union`
  - `intersection`
  - `difference`
- Comparing multiple datasets
- Finding:
  - unique achievements
  - common achievements
  - missing achievements

---

### Exercise 4 — Inventory Master

- Using dictionaries to store key-value data
- Parsing structured input (`item:quantity`)
- Handling duplicates and invalid input
- Using:
  - `.keys()`
  - `.values()`
  - `.update()`
- Performing calculations on dictionary data:
  - total quantity
  - percentage representation
  - most and least abundant items

---

### Exercise 5 — Stream Wizard

- Using generators with `yield`
- Creating infinite data streams
- Using `next()` to consume generators
- Managing memory efficiently
- Iterating over generator outputs
- Creating event-driven data systems

---

### Exercise 6 — Data Alchemist

- Using list comprehensions
- Using dictionary comprehensions
- Filtering and transforming data in one line
- Calculating averages
- Filtering data based on conditions
- Combining transformation and filtering efficiently

## Key Concepts Mastered

- Lists: ordered and mutable collections
- Tuples: ordered and immutable collections
- Sets: unordered collections of unique elements
- Dictionaries: key-value mappings
- Generators: memory-efficient data streams
- Comprehensions: concise data transformation syntax
- Error handling: robust input validation
